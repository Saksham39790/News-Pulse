# NewsPulse

A modern news aggregation web application built with Node.js, Express, JavaScript and NewsAPI.

## Features
- Real-time news headlines
- Search functionality
- Category filtering
- Dark mode
- Pagination
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- NewsAPI

## Installation

npm install
node server.js

Visit:
http://localhost:3000
